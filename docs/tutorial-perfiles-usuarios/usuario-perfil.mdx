# Asignación de usuario a perfil

<ImageDiv src="/img/work/sys/users/edit-profile-users.png" alt="Edit profile user" />

Para asignar un usuario a un perfil, es necesario seguir los siguientes pasos:

1. Ingresar a la vista de usuarios.
2. Seleccionar el usuario al que se le asignará el perfil.
3. Hacer clic en el botón de edición.
4. Seleccionar el perfil que se le asignará al usuario.
5. Guardar los cambios.
