# Paso 2: Creación de atributos de expedientes

Los atributos de expedientes son importantes para poder enriquecer a la estructura de expediente de información adicional. Si no se cuenta con atributos, es necesario crearlos para poder asignarlos a diversas estructuras de expediente.

A continuación, se da un ejemplo de cómo crear un atributo de expediente.

## Modal de creación

Para crear un atributo de expediente, se debe hacer clic en el botón `Agregar`, `Nuevo` o similar que se encuentra en la parte superior derecha de la vista, a continuación se abrirá un modal como el siguiente:

<ImageDiv src="/img/work/opr/attrs-expedientes/form-new-attr-expediente-1.png" alt="Nuevo atributo de expediente" />

Hay 4 tipos de dato que se pueden asignar a un atributo de expediente, en la sección [Creación de Atributos de expedientes](/docs/area-operacion/atributos-expedientes#creación) se explica cada uno de ellos.

Supongamos que creamos un nuevo atributo llamado `¿Es documento legal?`, el tipo de dato seleccionado es `Decisión`, se debe seleccionar el tipo de dato, ingresar la descripción y, seguún el tipo de dato, agregar los demás valores solicitados, en este caso, el `valor negativo` y el `valor positivo`, y dar clic en el botón `Guardar`.

<ImageDiv src="/img/tutorial/expediente/form-new-attr-exp.png" alt="Nuevo atributo de expediente" />

## Actualización en tiempo real

En tiempo real, se mostrará el atributo de expediente creado en la tabla.

Una vez creado el atributo de expediente, se procede a la creación de estructura de expediente.
